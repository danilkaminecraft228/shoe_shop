# Информационная система ООО «Обувь»

Desktop-приложение для магазина обуви. Демонстрационный экзамен по специальности 09.02.07 «Информационные системы и программирование».

## Технологии

- **Python 3**+
- **PyQt6** — графический интерфейс
- **MySQL 8** — йбаза данных
- **mysql-connector-python** — подключение к БД
- **Pillow** — обработка изображений
- **openpyxl** — импорт из Excel

## Установка и запуск

### 1. Клонирование / открытие проекта

Откройте папку `shoe_shop` в PyCharm или другой IDE.

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Создание базы данных MySQL

Запустите MySQL сервер (например, через XAMPP, MySQL Workbench или командную строку).

Выполните SQL-скрипт:

```bash
mysql -u root -p < schema.sql
```

Пароль для MySQL: **root** (по умолчанию).

### 4. Импорт данных

Положите Excel-файлы в папку `import_data/`:

- `Tovar.xlsx` — товары
- `user_import.xlsx` — пользователи
- `Заказ_import.xlsx` — заказы
- `Пункты выдачи_import.xlsx` — пункты выдачи

Затем запустите скрипт импорта:

```bash
python seed_import.py
```

### 5. Запуск приложения

```bash
python main.py
```

## Структура проекта

```
shoe_shop/
├── main.py                    # Точка входа
├── database.py                # Подключение к MySQL
├── schema.sql                 # Схема базы данных
├── seed_import.py             # Импорт данных из Excel
├── requirements.txt           # Зависимости
├── README.md                  # Документация (этот файл)
├── algorithm_description.md   # Алгоритм работы приложения
├── import_data/               # Excel-файлы для импорта
│   ├── Tovar.xlsx
│   ├── user_import.xlsx
│   ├── Заказ_import.xlsx
│   └── Пункты выдачи_import.xlsx
├── resources/                 # Ресурсы приложения
│   ├── Icon.ico               # Иконка приложения
│   ├── Icon.png               # Логотип
│   ├── picture.png            # Заглушка фото
│   └── products/              # Фото товаров (1.jpg … 10.jpg)
├── ui/                        # Окна приложения
│   ├── __init__.py
│   ├── login_window.py        # Окно авторизации
│   ├── products_window.py     # Список товаров
│   ├── product_form_window.py # Добавление/редактирование товара
│   ├── orders_window.py       # Список заказов
│   └── order_form_window.py   # Добавление/редактирование заказа
└── utils/                     # Вспомогательные модули
    ├── __init__.py
    ├── style.py               # Стили и константы
    └── db_helpers.py          # Функции работы с БД
```
