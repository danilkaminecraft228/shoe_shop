## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Создание базы данных

Запустите MySQL-сервер и выполните скрипт создания схемы:

```bash
mysql -u root < schema.sql
```

> **Важно:** пароль MySQL — пустая строка (`""`). Если у вас установлен пароль, укажите его в `database.py`.

### 3. Импорт данных (опционально)

Поместите Excel-файлы в папку `import_data/` и запустите:

```bash
python seed_import.py
```

### 4. Запуск приложения

```bash
python main.py
```

## 🏗️ Архитектура

```
shoe_shop/
├── main.py                    # Точка входа, навигация между окнами
├── database.py                # Подключение к MySQL
├── schema.sql                 # Схема БД (10 таблиц, 3NF)
├── seed_import.py             # Импорт из Excel в БД
├── requirements.txt           # Зависимости
├── import_data/               # Excel-файлы для импорта
├── resources/                 # Иконки, логотип, фото товаров
├── ui/                        # Окна приложения (PyQt6)
│   ├── login_window.py        # Авторизация
│   ├── products_window.py     # Список товаров (карточки)
│   ├── product_form_window.py # Добавление/редактирование товара
│   ├── orders_window.py       # Список заказов (таблица)
│   └── order_form_window.py   # Добавление/редактирование заказа
└── utils/                     # Вспомогательные модули
    ├── style.py               # Цветовая схема, шрифты, стили
    └── db_helpers.py          # 16 функций запросов к БД
```

### База данных (10 таблиц)

| Таблица | Назначение |
|---------|-----------|
| `roles` | Роли пользователей (4 шт.) |
| `users` | Пользователи системы |
| `categories` | Категории товаров |
| `manufacturers` | Производители |
| `suppliers` | Поставщики |
| `products` | Товары (с FK на категории, производителя, поставщика) |
| `pickup_points` | Пункты выдачи |
| `order_statuses` | Статусы заказов |
| `orders` | Заказы (с FK на клиента, пункт выдачи, статус) |
| `order_items` | Составы заказов (связь M:N) |

## 📋 Особенности реализации

- **Параметризованные SQL-запросы** — защита от SQL-инъекций
- **Debounce-поиск** (300 мс) — поиск товаров в реальном времени без лишних запросов
- **Кастомные стили** — тёмно-зелёная цветовая схема через `utils/style.py`
- **Валидация** — проверка данных перед сохранением (цен, скидок, обязательных полей)
- **Импорт из Excel** — 4 вида файлов с обработкой дат, дубликатов и ссылочной целостности
- **4 роли доступа** — от гостя до администратора с разграничением прав
